Webserv Test Dashboard

Green-themed validation page for the first virtual server on port 8080.

Port
8080
Server ID
#1
Host
localhost
Protocol
HTTP/1.1

1. Static Website

GET

Serve static files such as HTML, CSS, images, and JavaScript.

Open static page
curl -i http://localhost:8080/static/about.html

2. HTTP Methods

GET / POST / DELETE

Upload a file with POST, inspect the upload directory with GET, then remove a file with DELETE.

Upload with POST
curl -i -F "file=@./test.txt" http://localhost:8080/upload/
View uploads with GET
Open upload directory
curl -i http://localhost:8080/upload/
Delete a file with DELETE
curl -i -X DELETE http://localhost:8080/upload/test.txt

3. Method Restriction

405

The /methods/ route allows GET only. This POST request should return 405.

curl -i -X POST http://localhost:8080/methods/

4. Directory Listing

autoindex

Show the generated listing for the configured directory.

Open listing
curl -i http://localhost:8080/listing/

5. Redirection

302

Visit the old site route and verify that it redirects to the new site.

Open old site
curl -i -L http://localhost:8080/old-site/

6. Custom Error Page

404

Request a missing path and confirm the custom 404 page is returned.

Trigger 404
curl -i http://localhost:8080/this-does-not-exist

7. CGI

GET / POST

Run CGI scripts with query strings, POST bodies, environment output, and timeout behavior.

CGI GET
Open CGI GET
curl -i "http://localhost:8080/cgi-bin/hello.py?name=evaluator"
CGI POST
curl -i -X POST -d "msg=hello cgi" http://localhost:8080/cgi-bin/hello.py
CGI environment
Open environment dump
curl -i http://localhost:8080/cgi-bin/env_dump.py
CGI timeout
Open timeout test
curl -i http://localhost:8080/cgi-bin/infinite_loop.py

8. Cookies & Session

bonus

Refresh the counter script and verify that the server keeps the same session.

Open session counter
curl -i -c cookies.txt -b cookies.txt http://localhost:8080/cgi-bin/counter.py

port 8081
